# Cyber-Minimalist Portfolio Website

A high-performance, responsive, and "Cyber-Minimalist" single-page portfolio website built to showcase professional background in Computer Science, AI Robotics, and Data Science.

## Features

- **Cyber-Minimalist Design:** Modern interface featuring glassmorphism and smooth animations.
- **Responsive Layout:** Fully responsive design that works seamlessly across all devices.
- **Framer Motion Animations:** Interactive elements and scroll animations.
- **Tech Stack:** Built with React 19, TypeScript, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
